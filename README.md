## Description

A powerful WooCommerce plugin that lets customers preview product details instantly through a custom-designed modal for reducing clicks, speeding up browsing, and creating a smoother, more engaging shopping experience.

- [Read Official Documentation](https://www.gethandytools.site/plugin/handy-quick-view-for-woocommerce/)
- [Try Live Demo Shop](https://www.handytoolsdemo.shop)

## Why You Should Use It?

Explore the comprehensive list of features and compelling reasons to choose Handy Quick View for your WooCommerce store.

- **Customizable Quick View Modal** - Create beautifully designed Quick View popups that match your brand. Adjust layout, spacing, colors, images, and content to give customers a seamless preview experience.
- **Instant Product Preview** - Let shoppers view essential product details images, price, rating, description, and variations without loading the full product page, making browsing much faster.
- **Image Zoom & Lightbox Support** - Let customers zoom in and view full-size product images in a lightbox directly from the Quick View modal, giving them a clearer look without leaving the page.
- **Add to Cart Inside Quick View** - Shoppers can add products to their cart without leaving the current page, creating a faster buying flow and reducing drop offs.
- **Keyboard & Click Navigation** - Customers can close the modal by clicking outside or pressing escape, ensuring intuitive usage for all types of shoppers.
- **Support Product Variation** - Allow customers to view and choose product variations (color, size, type) directly from the Quick View popup, improving convenience and boosting conversions.
- **AJAX-Powered Loading** - The Quick View modal loads instantly via AJAX, ensuring smooth, lag-free interactions that keep users engaged and reduce browsing friction.
- **Mobile-Responsive Modal** - The Quick View modal automatically adapts to all screen sizes phones, tablets, and desktops ensuring perfect usability across all devices.
- **Developer Hooks & Filters** - Built with clean code and extendable hooks that developers can use to customize or integrate the Quick View with other tools and features.

## Installation

### Manual Installation via FTP
- Download the plugin package to your computer.
- Extract the downloaded file (if it is in .zip format).
- Using your preferred FTP client, upload the extracted plugin folder to the `/wp-content/plugins/` directory of your WordPress installation.
- Log in to your WordPress dashboard.
- Go to Plugins → Installed Plugins and click Activate next to the plugin name.

### Manual Installation via WordPress Admin (Upload Method)
- Download the plugin archive (.zip file).
- Log in to your WordPress dashboard.
- Navigate to Plugins → Add New.
- Click the Upload Plugin button at the top of the page.
- Select the downloaded .zip file and click Install Now.
- After installation completes, click Activate to enable the plugin.
